Practice

Here are some instructions to practice working with a data set.

  1. Choose a data set from the list of csv files below, or go to kaggle and choose a different one.
  2. Download your chosen CSV file.
  3. Download the Jupyter notebook that corresponds with your data set. Make sure that both the CSV and Jupyter notebook files are in the same directory on your computer
  4. Launch the Jupyter notebook (through anaconda). Go through and complete the exercises.
  5. If you get stuck, use Google, our resource page, or look at a completed Jupyter notebook (below).

Note: You will need a kaggle account to download a data set. You will need python downloaded onto your computer. We recommend anaconda.

Useful Commands

  • For Cleaning Data
    • df.info() → Displays general info about dataframe (names of columns, number of entries per column, data type). Use to get a general sense of your data.
    • df.isnull().sum() → Lists total number of null values in each column of dataframe
    • df["ColumnName"].value_counts() → Lists all unique values and how many times they occur in specific column
    • df["ColumnName"].replace(to_replace, value, inplace=True) → Replace one or more values (to_replace) with another value (value)
    • df["ColumnName"].unique() → Returns array of all unique values in a specific column. Can be used to see if you need to replace the names of any values.
  • For Visualizing Data
    • df.groupby(["Column1", "Column2"])[["Column 3"]] → Group by a series of columns.
    • df.plot() → Makes plot of dataframe.
    • plt.plot(x, y) → Creates plot (using matplotlib, need plt.show() after)
