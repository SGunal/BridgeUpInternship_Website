What is Pandas and Why do we used?


Pandas is a python library which provides tools for data analysis. The Pandas library is not named after the animal, but after the econometric term “panel data” This library allowed us to look at, describe, and visualize big datasets in an organized manner. Some of the functions we used included describe, which finds the mean, median, and other values of a column, value_counts, which sorts values in a column by how many times they appear, groupby, which groups the dataframe by a selected column or columns, info, which lists the columns of a dataframe and what data type they contain, plot, which creates different types of graphs out of data, and many more. In addition, we learned how to create new dataframes out of the originals with more specific data. These functions help us clean up and better understand what our data contains. By understanding our data we can ask questions that can be answered by our data.
